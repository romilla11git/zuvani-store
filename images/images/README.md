# Product Images Folder

This folder contains all product images for Zuvani Tees.

## How to use:
- Upload product photos here
- Name them: tee1.jpg, polo1.jpg, hoodie1.jpg
- In admin panel, use path: images/your-photo-name.jpg

## Example:
- images/classic-tee-black.jpg
- images/premium-polo-white.jpg
- images/urban-hoodie-gray.jpg
